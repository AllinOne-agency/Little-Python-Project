# To-Do List Manager

## Description
This is a beginner-friendly project to practice basic Python programming skills such as lists, loops, and user input. The program allows users to:
- View their to-do list.
- Add tasks to the list.
- Remove tasks from the list.

## How to Run the Program
1. Make sure you have Python installed on your computer.
2. Copy the code into a Python file (e.g., `todo_list.py`).
3. Open your terminal or command prompt.
4. Navigate to the folder where you saved the file.
5. Run the program using the command: `python todo_list.py`.
